# frontPractice - 前端面试复习仓库

用于前端面试系统性复习，包含基础、JS、浏览器、框架、工程化、网络、算法等。

## 目录结构
- 01-HTML-CSS 基础与布局
- 02-JavaScript 核心与手写题
- 03-Browser-API 浏览器原理
- 04-Framework 框架（Vue/React）
- 05-Engineering 工程化
- 06-Computer-Network 计算机网络
- 07-Algorithm 算法与数据结构
- 08-Project 项目实战
- 09-Interview 面试题汇总
