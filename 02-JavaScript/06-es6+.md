# ES6+ 新特性核心基础知识

## 一、模块化（Module）

### 1. 什么是模块化？

- **模块化**是一种将代码分割成独立、可重用单元的方式
- ES6 引入了官方的模块化系统，解决了之前 CommonJS 和 AMD 的兼容性问题
- 模块化的核心是 `import` 和 `export` 语句

### 2. 导出（Export）

- **命名导出**：可以导出多个值，使用 `import { name } from 'module'` 导入
- **默认导出**：每个模块只能有一个默认导出，使用 `import name from 'module'` 导入
- **混合导出**：可以同时使用命名导出和默认导出

#### 命名导出示例

```javascript
// module.js
export const name = 'Module';
export const version = '1.0.0';

export function greet() {
  return `Hello from ${name}`;
}
```

#### 默认导出示例

```javascript
// module.js
const name = 'Module';
const version = '1.0.0';

function greet() {
  return `Hello from ${name}`;
}

export default {
  name,
  version,
  greet
};
```

### 3. 导入（Import）

- **命名导入**：导入指定的命名导出
- **默认导入**：导入默认导出
- **重命名导入**：使用 `as` 关键字重命名导入的模块
- **导入所有**：使用 `* as name` 导入所有命名导出

#### 导入示例

```javascript
// 命名导入
import { name, version, greet } from './module.js';

// 默认导入
import module from './module.js';

// 重命名导入
import { name as moduleName, greet as sayHello } from './module.js';

// 导入所有
import * as module from './module.js';
```

### 4. 动态导入

- 使用 `import()` 函数动态加载模块
- 返回一个 Promise，解析为模块的导出对象
- 适用于按需加载和代码分割

```javascript
// 动态导入
import('./module.js')
  .then(module => {
    console.log(module.name);
    module.greet();
  })
  .catch(error => {
    console.error('加载模块失败:', error);
  });

// 使用 async/await
async function loadModule() {
  try {
    const module = await import('./module.js');
    console.log(module.name);
    module.greet();
  } catch (error) {
    console.error('加载模块失败:', error);
  }
}
```

### 5. 模块的特性

- **严格模式**：模块默认在严格模式下运行
- **独立作用域**：模块有自己的作用域，不会污染全局作用域
- **静态分析**：导入和导出在编译时处理，支持静态分析
- **循环依赖**：支持模块间的循环依赖

## 二、解构赋值（Destructuring）

### 1. 什么是解构赋值？

- **解构赋值**是一种从数组或对象中提取值并赋给变量的语法
- 可以简化代码，提高可读性
- 支持数组解构和对象解构

### 2. 数组解构

- 按照位置提取数组元素
- 支持默认值和剩余参数

```javascript
// 基本用法
const [a, b, c] = [1, 2, 3];
console.log(a, b, c); // 输出: 1 2 3

// 跳过元素
const [x, , y] = [4, 5, 6];
console.log(x, y); // 输出: 4 6

// 默认值
const [p = 10, q = 20] = [30];
console.log(p, q); // 输出: 30 20

// 剩余参数
const [first, ...rest] = [1, 2, 3, 4, 5];
console.log(first); // 输出: 1
console.log(rest); // 输出: [2, 3, 4, 5]
```

### 3. 对象解构

- 按照属性名提取对象属性
- 支持重命名、默认值和剩余属性

```javascript
// 基本用法
const { name, age } = { name: 'John', age: 30 };
console.log(name, age); // 输出: John 30

// 重命名
const { name: username, age: userAge } = { name: 'John', age: 30 };
console.log(username, userAge); // 输出: John 30

// 默认值
const { name, age = 25 } = { name: 'John' };
console.log(name, age); // 输出: John 25

// 剩余属性
const { name, ...rest } = { name: 'John', age: 30, city: 'Beijing' };
console.log(name); // 输出: John
console.log(rest); // 输出: { age: 30, city: 'Beijing' }
```

### 4. 嵌套解构

- 可以嵌套使用数组解构和对象解构
- 处理复杂的数据结构

```javascript
// 嵌套数组解构
const [a, [b, c]] = [1, [2, 3]];
console.log(a, b, c); // 输出: 1 2 3

// 嵌套对象解构
const { user: { name, age } } = { user: { name: 'John', age: 30 } };
console.log(name, age); // 输出: John 30

// 混合嵌套
const [a, { name, age }] = [1, { name: 'John', age: 30 }];
console.log(a, name, age); // 输出: 1 John 30
```

### 5. 函数参数解构

- 在函数参数中使用解构赋值
- 简化函数参数的处理

```javascript
// 数组参数解构
function sum([a, b]) {
  return a + b;
}
console.log(sum([1, 2])); // 输出: 3

// 对象参数解构
function greet({ name, age = 25 }) {
  return `Hello, ${name}! You are ${age} years old.`;
}
console.log(greet({ name: 'John' })); // 输出: Hello, John! You are 25 years old.

// 混合使用
function getData([first, second], { name }) {
  return `${name} has ${first} and ${second}`;
}
console.log(getData([1, 2], { name: 'John' })); // 输出: John has 1 and 2
```

## 三、箭头函数（Arrow Functions）

### 1. 什么是箭头函数？

- **箭头函数**是 ES6 引入的一种新的函数语法
- 使用 `=>` 符号定义函数，语法更简洁
- 没有自己的 `this`、`arguments`、`super` 或 `new.target`

### 2. 基本语法

- **单个参数**：`param => expression`
- **多个参数**：`(param1, param2) => expression`
- **无参数**：`() => expression`
- **多行表达式**：`() => { statements }`

```javascript
// 单个参数
const double = x => x * 2;
console.log(double(5)); // 输出: 10

// 多个参数
const sum = (a, b) => a + b;
console.log(sum(1, 2)); // 输出: 3

// 无参数
const greet = () => 'Hello!';
console.log(greet()); // 输出: Hello!

// 多行表达式
const factorial = n => {
  if (n <= 1) return 1;
  return n * factorial(n - 1);
};
console.log(factorial(5)); // 输出: 120
```

### 3. 箭头函数的特性

- **没有自己的 this**：箭头函数的 `this` 指向定义时的词法作用域
- **没有 arguments 对象**：可以使用剩余参数 `...args` 代替
- **不能作为构造函数**：不能使用 `new` 关键字调用
- **没有 prototype**：箭头函数没有 `prototype` 属性
- **不能使用 yield**：箭头函数不能作为生成器函数

### 4. this 绑定

- 箭头函数的 `this` 继承自外层作用域
- 解决了传统函数中 `this` 指向不明确的问题
- 适用于回调函数和事件处理程序

```javascript
// 传统函数的 this 问题
const obj = {
  name: 'John',
  greet: function() {
    setTimeout(function() {
      console.log(`Hello, ${this.name}!`); // this 指向全局对象
    }, 1000);
  }
};
obj.greet(); // 输出: Hello, undefined!

// 箭头函数解决 this 问题
const obj2 = {
  name: 'John',
  greet: function() {
    setTimeout(() => {
      console.log(`Hello, ${this.name}!`); // this 继承自 greet 方法
    }, 1000);
  }
};
obj2.greet(); // 输出: Hello, John!
```

### 5. 箭头函数的应用场景

- **回调函数**：简化回调函数的写法
- **数组方法**：与 `map`、`filter`、`reduce` 等方法配合使用
- **简洁表达式**：适合单行表达式的函数
- **避免 this 绑定问题**：在需要保持 `this` 上下文的场景中使用

```javascript
// 数组方法
const numbers = [1, 2, 3, 4, 5];

// map
const doubled = numbers.map(x => x * 2);
console.log(doubled); // 输出: [2, 4, 6, 8, 10]

// filter
const even = numbers.filter(x => x % 2 === 0);
console.log(even); // 输出: [2, 4]

// reduce
const sum = numbers.reduce((acc, x) => acc + x, 0);
console.log(sum); // 输出: 15

// 回调函数
setTimeout(() => console.log('Hello!'), 1000);

// 事件处理
document.querySelector('button').addEventListener('click', () => {
  console.log('Button clicked!');
});
```

## 四、其他 ES6+ 新特性

### 1. 模板字符串（Template Literals）

- 使用反引号（`）定义字符串
- 支持多行字符串
- 支持变量插值（`${expression}`）

```javascript
// 基本用法
const name = 'John';
const greeting = `Hello, ${name}!`;
console.log(greeting); // 输出: Hello, John!

// 多行字符串
const message = `
  Hello,
  ${name}!
  How are you?
`;
console.log(message);

// 表达式
const a = 10;
const b = 20;
const result = `The sum of ${a} and ${b} is ${a + b}.`;
console.log(result); // 输出: The sum of 10 and 20 is 30.
```

### 2. 扩展运算符（Spread Operator）

- 使用 `...` 符号
- 可以展开数组、对象等可迭代对象
- 用于数组拷贝、合并和函数参数

```javascript
// 数组展开
const arr1 = [1, 2, 3];
const arr2 = [...arr1, 4, 5];
console.log(arr2); // 输出: [1, 2, 3, 4, 5]

// 对象展开
const obj1 = { a: 1, b: 2 };
const obj2 = { ...obj1, c: 3, d: 4 };
console.log(obj2); // 输出: { a: 1, b: 2, c: 3, d: 4 }

// 函数参数
function sum(a, b, c) {
  return a + b + c;
}
const numbers = [1, 2, 3];
console.log(sum(...numbers)); // 输出: 6
```

### 3. 剩余参数（Rest Parameters）

- 使用 `...args` 语法
- 收集函数的剩余参数为一个数组
- 只能作为函数的最后一个参数

```javascript
// 基本用法
function sum(...numbers) {
  return numbers.reduce((acc, num) => acc + num, 0);
}
console.log(sum(1, 2, 3, 4, 5)); // 输出: 15

// 与其他参数结合
function greet(name, ...messages) {
  console.log(`Hello, ${name}!`);
  messages.forEach(message => console.log(message));
}
greet('John', 'How are you?', 'Nice to meet you!');
// 输出:
// Hello, John!
// How are you?
// Nice to meet you!
```

### 4. 类（Class）

- ES6 引入了 `class` 关键字，提供了更简洁的面向对象编程语法
- 支持继承、静态方法、getter/setter 等特性
- 本质上是原型继承的语法糖

```javascript
// 基本类
class Person {
  constructor(name, age) {
    this.name = name;
    this.age = age;
  }

  greet() {
    return `Hello, I'm ${this.name}, ${this.age} years old.`;
  }

  static create(name, age) {
    return new Person(name, age);
  }
}

// 继承
class Student extends Person {
  constructor(name, age, grade) {
    super(name, age);
    this.grade = grade;
  }

  study() {
    return `${this.name} is studying in grade ${this.grade}.`;
  }
}

// 使用
const person = new Person('John', 30);
console.log(person.greet()); // 输出: Hello, I'm John, 30 years old.

const student = new Student('Jane', 15, 9);
console.log(student.greet()); // 输出: Hello, I'm Jane, 15 years old.
console.log(student.study()); // 输出: Jane is studying in grade 9.

const person2 = Person.create('Bob', 25);
console.log(person2.greet()); // 输出: Hello, I'm Bob, 25 years old.
```

### 5. 增强的对象字面量（Enhanced Object Literals）

- 简化对象字面量的写法
- 支持简写属性、方法、计算属性名等

```javascript
// 简写属性
const name = 'John';
const age = 30;
const person = { name, age };
console.log(person); // 输出: { name: 'John', age: 30 }

// 简写方法
const obj = {
  greet() {
    return 'Hello!';
  }
};
console.log(obj.greet()); // 输出: Hello!

// 计算属性名
const key = 'name';
const obj2 = {
  [key]: 'John',
  [`age_${2023 - 1990}`]: 33
};
console.log(obj2); // 输出: { name: 'John', age_33: 33 }
```

### 6. 新的字符串方法

- `includes()`：检查字符串是否包含指定值
- `startsWith()`：检查字符串是否以指定值开头
- `endsWith()`：检查字符串是否以指定值结尾
- `repeat()`：重复字符串指定次数

```javascript
const str = 'Hello, world!';

console.log(str.includes('world')); // 输出: true
console.log(str.startsWith('Hello')); // 输出: true
console.log(str.endsWith('!')); // 输出: true
console.log('a'.repeat(3)); // 输出: aaa
```

### 7. 新的数组方法

- `find()`：返回第一个满足条件的元素
- `findIndex()`：返回第一个满足条件的元素的索引
- `includes()`：检查数组是否包含指定值
- `flat()`：扁平化数组
- `flatMap()`：先映射后扁平化

```javascript
const numbers = [1, 2, 3, 4, 5];

console.log(numbers.find(n => n > 3)); // 输出: 4
console.log(numbers.findIndex(n => n > 3)); // 输出: 3
console.log(numbers.includes(3)); // 输出: true

const nested = [1, [2, [3, 4]]];
console.log(nested.flat()); // 输出: [1, 2, [3, 4]]
console.log(nested.flat(2)); // 输出: [1, 2, 3, 4]

const words = ['hello', 'world'];
console.log(words.flatMap(word => word.split(''))); // 输出: ['h', 'e', 'l', 'l', 'o', 'w', 'o', 'r', 'l', 'd']
```

### 8. 新的数字特性

- `Number.isFinite()`：检查值是否为有限数字
- `Number.isNaN()`：检查值是否为 NaN
- `Number.isInteger()`：检查值是否为整数
- `Number.EPSILON`：表示 JavaScript 中最小的正数

```javascript
console.log(Number.isFinite(123)); // 输出: true
console.log(Number.isFinite(Infinity)); // 输出: false
console.log(Number.isNaN(NaN)); // 输出: true
console.log(Number.isInteger(42)); // 输出: true
console.log(Number.isInteger(42.5)); // 输出: false
console.log(Number.EPSILON); // 输出: 2.220446049250313e-16
```

### 9. 新的 Symbol 类型

- `Symbol` 是一种新的原始数据类型
- 用于创建唯一的标识符
- 可以作为对象的属性键

```javascript
// 创建 Symbol
const sym1 = Symbol('description');
const sym2 = Symbol('description');
console.log(sym1 === sym2); // 输出: false

// 作为对象属性
const obj = {
  [sym1]: 'value'
};
console.log(obj[sym1]); // 输出: value

// 内置 Symbol
console.log(Symbol.iterator); // 输出: Symbol(Symbol.iterator)
console.log(Symbol.toStringTag); // 输出: Symbol(Symbol.toStringTag)
```

### 10. 可选链操作符（Optional Chaining）

- 使用 `?.` 操作符
- 当访问嵌套对象的属性时，如果中间属性不存在，不会抛出错误
- 适用于处理可能不存在的属性

```javascript
const user = {
  name: 'John',
  address: {
    city: 'Beijing'
  }
};

// 传统方式
console.log(user && user.address && user.address.city); // 输出: Beijing

// 可选链
console.log(user?.address?.city); // 输出: Beijing
console.log(user?.profile?.age); // 输出: undefined
```

### 11. 空值合并操作符（Nullish Coalescing Operator）

- 使用 `??` 操作符
- 当左侧表达式为 `null` 或 `undefined` 时，返回右侧表达式
- 与 `||` 不同，`0`、`''`、`false` 等 falsy 值不会触发右侧表达式

```javascript
// 空值合并
const value = null ?? 'default';
console.log(value); // 输出: default

const value2 = 0 ?? 'default';
console.log(value2); // 输出: 0（因为 0 不是 null 或 undefined）

// 与 || 对比
const value3 = 0 || 'default';
console.log(value3); // 输出: default（因为 0 是 falsy 值）
```

### 12. 逻辑赋值操作符

- 结合逻辑运算符和赋值操作符
- 包括 `&&=`、`||=`、`??=`

```javascript
// &&=
let a = 1;
a &&= 2;
console.log(a); // 输出: 2（因为 a 是 truthy 值）

// ||=
let b = 0;
b ||= 2;
console.log(b); // 输出: 2（因为 b 是 falsy 值）

// ??=
let c = null;
c ??= 2;
console.log(c); // 输出: 2（因为 c 是 null）

let d = 0;
d ??= 2;
console.log(d); // 输出: 0（因为 d 不是 null 或 undefined）
```

## 六、总结

- **模块化**：`import` 和 `export` 语句

- **模板字符串**：支持多行和变量插值
- **扩展运算符**：展开数组和对象
- **剩余参数**：收集函数参数
- **类**：面向对象编程的语法糖
- **增强的对象字面量**：简化对象定义
- **新的数组和字符串方法**：提供更丰富的功能
- **可选链和空值合并**：处理可能不存在的值

### 2. 优势

- **代码更简洁**：箭头函数、模板字符串等特性减少了样板代码
- **可读性更高**：解构赋值、模块化等特性使代码结构更清晰
- **功能更强大**：新的方法和操作符提供了更多功能
- **开发效率更高**：语法糖和新特性简化了开发过程

### 3. 注意事项

- **兼容性**：需要考虑旧浏览器的支持
- **性能**：某些新特性可能有性能影响
- **过度使用**：避免过度使用新特性，保持代码清晰
- **学习曲线**：需要时间学习和适应新特性

### 4. 未来发展

- ES6+ 标准持续更新，不断引入新特性
- 浏览器和 Node.js 对新特性的支持越来越好
- 工具链（如 Babel、webpack）使新特性的使用更加便捷

通过掌握 ES6+ 的核心新特性，可以编写更现代、更简洁、更高效的 JavaScript 代码，提高开发效率和代码质量。
